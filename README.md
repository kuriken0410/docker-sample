## :globe_with_meridians: Todoリストマネージャーアプリケーションとは
- Dockerに乗せているアプリケーションです。
- 実施したいことを登録することで、登録したタスクをリスト管理できます。

## :globe_with_meridians: 機能一覧
- 実施したいタスクの登録機能。
- 登録したタスクの削除機能。
